**CMDrawer** - CMDrawer is a simple paint-like console program made with Microsoft Assembly Language (masm32).

**Features:**

**路** Use your mouse (touchpad) to draw ascii-graphics in working area. You can change active symbol for drawing, its size and color in Tools menu.

**路** Clear working area if needed

**路** Export your drawing to .txt file (not done)

**路** Use some other tools (not done)

Current version: 1.3.2
